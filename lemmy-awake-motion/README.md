# Lemmy Awake Motion

Canvas-based awake motion prototype for a 1280 x 1920 vertical screen.

## Files

- `index.html`: page markup
- `style.css`: fullscreen responsive canvas layout
- `script.js`: animated gradient volume motion

## Preview

Open `index.html` in a browser. The canvas keeps a 1280 x 1920 aspect ratio and scales to the viewport.

Click or tap the screen to restart the awake motion.
