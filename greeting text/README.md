# Hangul Handwriting Animation

Figma SVG artwork with a looped Hangul handwriting animation.

## Files

- `index.html`: Page markup
- `style.css`: Layout and visual styles
- `script.js`: Hangul stroke-order animation and loop timing
- `group.svg`: Original Figma artwork

## Run Locally

Open `index.html` in a browser.

## GitHub Pages

Upload all files in this folder to a GitHub repository, then enable GitHub Pages:

1. Go to repository `Settings`
2. Open `Pages`
3. Set source to the main branch root
4. Save

The page will be served from the GitHub Pages URL.
