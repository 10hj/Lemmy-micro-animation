# Falling Pills

A small HTML/CSS animation using the original `Group.svg` pill artwork.

The SVG paths are preserved and animated with CSS:

- each pill falls at a different speed
- each pill rotates while falling
- staggered delays create a continuous falling motion
- no JavaScript is required

## Files

- `index.html`: main page with the inline animated SVG
- `style.css`: animation and layout styles
- `Group.svg`: original source SVG

## Usage

Open `index.html` in a browser.
