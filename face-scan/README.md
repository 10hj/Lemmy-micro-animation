# Face Scan Motion

A small HTML/CSS face recognition animation using the original `Union.svg` face artwork.

The SVG face shape is preserved and animated with CSS:

- a scanning line passes over the face
- corner brackets pulse like a recognition frame
- subtle rings and dots create an active scanning state
- no JavaScript is required

## Files

- `index.html`: main page with the inline animated SVG
- `style.css`: animation and layout styles
- `Union.svg`: original source SVG

## Usage

Open `index.html` in a browser.
