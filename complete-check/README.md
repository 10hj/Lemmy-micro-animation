# Complete Check

A small HTML/CSS completion animation using the original `Group.svg` check mark artwork.

The SVG check shape is preserved and animated with CSS:

- the circle draws in first
- the check appears with a soft pop
- small sparks reinforce the completed state
- no JavaScript is required

## Files

- `index.html`: main page with the inline animated SVG
- `style.css`: animation and layout styles
- `Group.svg`: original source SVG

## Usage

Open `index.html` in a browser.
