# Coffee Steam

A small HTML/CSS animation using the original coffee artwork.

The SVG paths are preserved and animated with CSS:

- three steam shapes rise, drift, and fade
- the original steam and cup colors are preserved
- the cup has a subtle breathing motion
- no JavaScript is required

## Files

- `index.html`: main page with the inline animated SVG
- `style.css`: animation and layout styles

## Usage

Open `index.html` in a browser.
