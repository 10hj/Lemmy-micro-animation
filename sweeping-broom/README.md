# Sweeping Broom

A small HTML/CSS animation using the original `Group.svg` broom artwork.

The SVG paths are preserved and animated with CSS:

- the broom sweeps side to side
- the dust shapes scatter and fade
- no JavaScript is required

## Files

- `index.html`: main page with the inline animated SVG
- `style.css`: animation and layout styles
- `Group.svg`: original source SVG

## Usage

Open `index.html` in a browser.
