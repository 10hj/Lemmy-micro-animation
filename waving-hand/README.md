# Waving Hand

A small HTML/CSS animation that uses the original `Vector.svg` shape and applies a waving motion.

## Files

- `index.html`: main page with the animation
- `Vector.svg`: source SVG used as the hand shape

## How to Use

Open `index.html` in a browser.

For GitHub Pages, upload this folder to a repository and serve it from the repository root or from the `docs` folder.
