# Voice Motion

A small HTML/CSS speaking voice animation using the original `Group.svg` waveform artwork.

The SVG bar shapes are preserved and animated with CSS:

- center and side bars pulse at different rhythms
- each bar scales vertically like active speech
- no JavaScript is required

## Files

- `index.html`: main page with the inline animated SVG
- `style.css`: animation and layout styles
- `Group.svg`: original source SVG

## Usage

Open `index.html` in a browser.
